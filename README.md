# 记事本-后端程序(Nest.js)

毕设项目，大缝合怪，人生第一个写完的Node.js项目，很多特性没用上，只是能跑起来

**需要配合notepad前端才能使用**

## 已用组件：
- Nest.js (主框架)
- typeorm (数据库ORM)
- bcryptjs (密码加密)
- mysql2 (数据库连接)
- passport (登录验证，用户名密码 + JWT两种方式)

## 加载各种库

```bash
$ npm install pnpm -g
$ pnpm install
```

## 如何运行：

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
